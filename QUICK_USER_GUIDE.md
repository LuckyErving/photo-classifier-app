# 快速使用指南

## 📸 新功能：文件夹管理

### 基本操作

#### 1️⃣ 选择文件夹
拍照界面顶部有一个文件夹选择器，点击可以选择要保存照片的文件夹。

```
┌─────────────────────┬───┐
│  选择文件夹 ▼       │ + │
└─────────────────────┴───┘
```

#### 2️⃣ 创建新文件夹
点击右侧的 **"+"** 按钮，输入文件夹名称，即可创建新文件夹。

**示例文件夹名称：**
- 工作照片
- 家庭照片
- 旅行-2025
- 学习笔记
- 扫描文档

#### 3️⃣ 拍照保存
选择文件夹后，拍的照片会自动保存到该文件夹。下次打开应用，会自动选择上次使用的文件夹。

### 📁 导出到电脑

#### 方法：USB数据线
1. 用USB数据线连接手机和电脑
2. 在手机上选择 **"文件传输"** 模式
3. 在电脑上打开手机存储
4. 找到路径：**内部存储/DCIM/PhotoClassifier/**
5. 复制需要的文件夹到电脑

#### 目录结构示例
```
内部存储/
└── DCIM/
    └── PhotoClassifier/
        ├── Default/          (默认文件夹)
        ├── 工作照片/
        ├── 家庭照片/
        └── 旅行-2025/
```

### ⚠️ 注意事项

1. **文件夹名称**：不能包含 `/` 或 `\` 字符
2. **重名检查**：不能创建已存在的文件夹名称
3. **权限授予**：首次使用需要授予相机和存储权限
4. **卸载保留**：即使卸载应用，照片也会保留在DCIM目录

### 🔍 常见问题

**Q: 为什么找不到照片？**
A: 照片保存在 `内部存储/DCIM/PhotoClassifier/` 目录下。

**Q: 可以在电脑上直接删除文件夹吗？**
A: 可以，通过USB连接后可以直接管理文件夹和照片。

**Q: 卸载应用后照片会丢失吗？**
A: 不会，照片保存在公共DCIM目录，卸载应用后仍然保留。

**Q: 如何批量导出照片？**
A: 通过USB连接电脑，直接复制整个文件夹即可。

**Q: 文件夹可以重命名吗？**
A: 当前版本不支持，可以通过USB在电脑上重命名。

### 🎯 最佳实践

1. **开始拍照前先选择文件夹**，避免事后整理
2. **合理规划文件夹结构**，便于长期管理
3. **定期通过USB导出**，做好照片备份
4. **使用有意义的文件夹名称**，方便日后查找
5. **不同项目使用不同文件夹**，保持照片整理有序
